## 💡 ملاحظة هامة للتعديل والتخصيص | Customization Note
إذا كنت ترغب في حذف أو تعطيل أي أمر لا يهمك داخل السكريبت:
1. قم بفتح ملف `win2x.txt` المرفق والتعديل عليه بحذف السطر الذي لا تريده.
2. احفظ الملف بعد التعديل (Save).
3. قم بتغيير امتداد الملف من (`win2x.txt`) إلى (`win2x.bat`).
4. شغّل الملف كمسؤول (**Run as Administrator**) لتطبيق الإعدادات الخاصة بك.

## 📑 وظائف السكريبت بالتفصيل (ما الذي يفعله بجهازك؟)

### 1️⃣ إصلاح ملفات النظام الأساسية
* `sfc /scannow` 🡪 فحص وإصلاح ملفات نظام الويندوز التالفة.
* `DISM ... RestoreHealth` 🡪 صيانة صورة النظام وإصلاح الملفات المستعصية عبر الإنترنت.
* `DISM ... StartComponentCleanup` 🡪 تنظيف وتطهير مخزن مكونات النظام وتوفير مساحة.

### 2️⃣ فحص القرص الصلب وإصلاح الإقلاع
* `chkdsk C:` 🡪 فحص القرص C وإصلاح أخطاء النظام والملفات بشكل إجباري.
* `bootrec /rebuildbcd` 🡪 إعادة بناء ملفات الإقلاع لحل مشاكل إقلاع الويندوز.

### 3️⃣ تسريع الإنترنت وتنظيف الشبكة
* `netsh winsock reset` 🡪 إعادة ضبط بروتوكولات الشبكة بالكامل لحل مشاكل الاتصال.
* `ipconfig /flushdns` 🡪 تفريغ كاش الـ DNS لتسريع تصفح المواقع وحل مشاكل انقطاع الإنترنت.

### 4️⃣ تنظيف شامل للملفات المؤقتة والمهملات
* `cleanmgr` 🡪 تشغيل أداة تنظيف القرص الذكية التابعة للويندوز.
* `del Prefetch & Temp` 🡪 حذف ملفات الكاش والملفات المؤقتة المتراكمة التي تبطئ النظام.
* `wsreset -i` 🡪 إعادة تنشيط وتنظيف كاش متجر تطبيقات مايكروسوفت (Microsoft Store).
* `PowerShell GC Collect` 🡪 تحرير ذاكرة الوصول العشوائي (RAM) وتفريغها من العمليات العالقة.

### 5️⃣ تعزيز الخصوصية وإيقاف التجسس
* `AllowTelemetry = 0` 🡪 إيقاف إرسال بياناتك وتحركاتك إلى خوادم مايكروسوفت.
* `DiagTrack = disabled` 🡪 إيقاف خدمة التتبع والتشخيص التلقائي لتوفر لابتوبك/كمبيوترك أقصى سرعة.

### 6️⃣ تشغيل الأداء الأقصى وتحديث الواجهة
* `powercfg Ultimate` 🡪 تفعيل خطة الطاقة الخفية "الأداء الأقصى" (Ultimate Performance).
* `taskkill explorer` 🡪 إعادة تشغيل شريط المهام ومستكشف الملفات لتنشيط الويندوز وتطبيق التغييرات فوراً.

## 📖 طريقة الاستخدام الصحيحة
1. تأكد أن ملف السكريبت ينتهي بامتداد `.bat` (مثال: `win2x.bat`).
2. اضغط عليه كليك يمين ثم (**Run as Administrator**).
3. ستفتح لك شاشة سوداء، اتركها تماماً حتى تنتهي من كل الفحوصات والتنظيف وتغلق تلقائياً.
4. 💡 **[موصى به]** أعد تشغيل الجهاز فوراً بعد انتهاء السكريبت لضمان تطبيق تعديلات القرص الصلب والإقلاع بنجاح.
